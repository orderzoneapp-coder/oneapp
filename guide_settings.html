글로벌 환경설정 센터

NEXUS 생태계의 전역 비즈니스 룰을 통제하는 중앙 관제탑 가이드입니다.

1. 목적 및 역할 (Purpose)

  • 시스템의 통제탑 (Control Tower) NEXUS MerchOps 생태계 내 모든 앱(작업장, 마스터, 파서 등)이 공통으로 참조하는 전역 비즈니스 룰(Global Baseline)을 중앙에서 정의하고 통제합니다.
  • 운영 표준화 (Standardization) 작업자마다 다를 수 있는 마진율 계산법, 엑셀 헤더 식별법, 화면 레이아웃 등을 하나로 통일시켜 휴먼 에러를 원천 차단합니다.

2. 운영 플로우 (Operational Flow)

1

규칙 세팅

관리자가 환경설정 탭에 진입하여 클라우드 URL, 마진 룰(나누기/곱하기), 엑셀 매핑 키워드를 세팅합니다.

2

안전 저장 (F8)

[전역 설정 반영] 버튼 클릭 시, 시스템은 마진율 오류(0% 이하)를 내부 검증한 후 안전하게 브라우저 DB에 저장합니다.

3

실시간 전파 (Broadcast)

저장이 완료되는 즉시, 보이지 않는 백그라운드 무전(SYNC_TRIGGER)이 전체 시스템으로 발송됩니다.

4

전사 동기화

현재 열려있는 모든 작업 창(시세 관리, 마스터 DB 등)이 새로고침 없이 즉각 새로운 룰을 적용하여 작동합니다.

3. 다른 앱과의 유기적 연동

시세 관리 (작업장)

  • 단가 연산: 설정된 마진 룰을 읽어 엑셀 단가에 적용.
  • 엑셀 파싱: 제각각인 거래처 엑셀 헤더를 시스템 표준으로 치환.
  • UI 노출: 작업 뷰 노출 설정에 따라 필요 컬럼만 동적 표시.

마스터 DB (관제탑)

  • 클라우드 통신: API URL을 참조하여 클라우드 서버와 데이터 송수신.
  • UI 노출: 48개 속성 중 필요한 정보만 선별하여 화면에 표시.

스마트 파서 (분석기)

  • 데이터 매핑: 비정형 텍스트를 쪼갤 때 매핑 사전을 참조하여 알맞은 속성에 자동 할당.

4. 주요 핵심 기능

Feature 01

하이브리드 단가 연산 엔진

창고조건(예: 01)과 단위조건(예: BOX)을 교차 매칭하여 각기 다른 마진율을 설정합니다.

단순 곱하기(×) 외에, 실무 도소매 마진 계산법인 역산 마진(나누기, ÷) 방식을 완벽 지원합니다. (예: 마진 10% 확보 시 '원가 ÷ 0.9')

1만 원 기준 작동 예시를 실시간 UI로 보여주어 관리자의 실수를 방지합니다.

Feature 02

무한 확장형 엑셀 매핑 사전

관리자가 "새로운 마스터 속성명"과 "매칭할 엑셀 키워드"를 무제한으로 사전에 추가할 수 있습니다.

특이한 포맷의 엑셀이 들어와도 개발자 도움 없이 현장에서 즉시 매핑 사전을 확장하여 대응 가능합니다.

Feature 03

투트랙 위험 구역 (Danger Zone)

시스템 꼬임 대비 2단계 초기화 기능입니다.

  • 옵션 1 (설정 초기화): 상품 데이터는 보존하고 마진/매핑 룰만 기본값으로 덮어씁니다.
  • 옵션 2 (DB 완전 삭제): 마스터 DB까지 영구 삭제하는 하드 리셋입니다. 보안을 위해 타이핑 검증을 거칩니다.

5. 기술 아키텍처 및 보안 요약

저장소 (Storage)

브라우저 내장 IndexedDB를 사용하며, 안정적인 오프라인 저장을 지원합니다.

UI 프레임워크

사용자 환경 구성을 위해 최신 표준 웹 기술과 유틸리티 CSS(Tailwind)를 활용합니다.

보안 (Security)

기본적인 우클릭 차단 및 단축키 제어를 통해 설정값 임의 조작을 방어합니다.

통신 (Sync)

Event-driven 아키텍처를 통해 탭 간의 설정 변경 사항을 즉시 동기화합니다.
