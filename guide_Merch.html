ONEAPP MerchOps 통합 사용자 가이드

이 가이드는 식자재 유통사와 식당 사장님을 연결하는 B2B 플랫폼의 운영 혁신, ONEAPP MerchOps 시스템의 완벽한 사용을 위해 제작되었습니다.

1

초기 도입 가이드

시스템의 철학과 기본 작동 원리를 이해하여, 기능의 100%를 활용하기 위한 필수 배경지식입니다.

1. 비즈니스 가치

이 시스템은 '업무 효율성 극대화'와 '수익 안정성 보장'을 위해 탄생했습니다.

  • 압도적인 업무 시간 단축 수기 타이핑을 없앴습니다. AI 파서와 단축키로 몇 시간 걸리던 업무를 몇 분 만에 종료합니다.
  • 휴먼 에러 차단 및 역마진 방어 창고/단위별 마진율이 자동 적용됩니다. 특히 매입가가 0원일 경우 출고가도 강제로 0원 처리하여 치명적인 영업 손실을 원천 통제합니다.
  • 시세 변동의 즉각적인 반영 매일 변하는 단가를 지연 없이 B2B 쇼핑몰 및 내부 ERP에 동기화하여 고객 신뢰도를 높입니다.
  • 적은 인력으로 고품질 유지 정상 데이터는 시스템에 맡기고, 관리자는 비정상적으로 튀는 '이상 항목(Anomaly)'에만 집중하여 수만 건의 데이터를 통제합니다.

2. 핵심 UX 철학

⌨️

Hands-free (마우스 제로)

마우스 이동으로 인한 지연을 없앴습니다. 숫자 입력 후 Enter나 방향키만 누르면 다음 에러 칸으로 부드럽게 이동합니다. 업무 마감은 오직 단축키로 직렬화했습니다.

👁️

Focus on Anomaly (이상점 집중)

수만 건을 다 볼 필요가 없습니다. 단가가 폭등/폭락하거나, 0원 누락, 불완전 매칭된 항목만 강렬한 색상으로 하이라이트됩니다. 무채색(정상)은 건너뛰십시오.

3. 4대 핵심 구역

  • 데이터 저장소 (창고): 브라우저 내부에 데이터를 보관하여 렉 없이 쾌적하게 사용.
  • 연산 엔진 (두뇌): 외부 텍스트를 마스터 상품과 매칭하고 단가 자동 연산.
  • 상태/동작 통제 (관리자): 클라우드 백업, 카탈로그 그룹 관리, 히스토리 기록.
  • 화면 및 디자인 (UI): 눈에 보이는 표와 버튼. 두뇌 구역과 분리되어 안전함.

4. 데이터 연동 구조

모든 데이터는 브라우저의 '로컬 스토리지(IndexedDB)'를 중앙 허브로 주고받습니다.

  • 클라우드 연동: 내 PC의 작업을 구글 클라우드에 백업(Push)하거나 불러오기(Pull).
  • 환경설정: 엑셀 매핑 규칙, 마진 룰 시스템 전체 공통 적용.
  • 다중 탭 연동: 파서 창에서 수정 시 메인 창이 새로고침 없이 백그라운드에서 갱신.

5. 핵심 알고리즘 요약

스마트 파서

불필요한 노이즈(마감, 품절 등)를 걸러내고 상품명, 규격, 가격을 정확히 분리합니다.

스마트 매칭

과거 연결 이력은 학습 사전(parserDict)에 저장되어 다음엔 100% 자동 매칭됩니다.

하이브리드 마진 연산

창고코드/판매단위에 부합하는 마진 룰을 찾아 출고가/소분 단가를 산출합니다.

다중 키워드 검색

띄어쓰기를 섞어도 상품명, 규격 가중치를 계산해 가장 연관성 높은 상품을 찾습니다.

2

실무 매뉴얼 (상시 열람용)

[필독] 마법의 마감 단축키

어떤 화면에서든 작업이 끝나면 아래 순서대로 단축키를 누릅니다.

F7

검토 및 프리뷰

인상/인하/신규 건수 요약 팝업을 띄워 1초 최종 점검

F8

마스터 저장 (Commit)

시스템 메인 두뇌에 영구 덮어쓰고 클라우드 동기화 대기

F9

엑셀 다운로드

쇼핑몰용, ERP용, 대조용 3개 시트 통합 엑셀 파일 즉시 추출

F10

이미지 생성

식당 사장님 카톡 전송용 모바일 최적화 단가표 이미지 생성

업무 플로우 A : 스마트 파서 (카톡/문자 시세 처리)

  1. 1

    입력 (Input)

    카톡 시세표 텍스트를 그대로 복사하여 붙여넣고 카탈로그 지정 후 분석 시작.

  2. 2

    자동 연산 (Process)

    시스템이 노이즈를 제거하고 마진 룰을 곱해 단가를 계산합니다.

  3. 3

    투트랙 검증 (색상을 확인하세요!)

    일치: 단가가 급변하여 색상이 칠해진 항목만 훑어보며 수정.
    유사/불일치: 검색창에서 수동 연결하거나 [ERP 신규 코드]로 즉시 등록.
    누락: 이번 시세에 없는 상품은 선택 후 '숨김(제외)' 처리.
  4. 4

    확정 및 산출

    F7 (프리뷰) ➡️ F8 (저장) ➡️ 메인 이동 ➡️ F9 (엑셀) ➡️ F10 (이미지)

업무 플로우 B : 메인 워크스페이스 (엑셀/그룹)

  1. 1

    데이터 호출

    유통사 엑셀 업로드 / 파서 작업물 호출 / 저장해둔 견적서 그룹 호출.

  2. 2

    대조 및 보정

    단가가 변동되어 색상이 바뀐 항목 확인. 직접 숫자를 수정하거나 [룰 적용] 버튼으로 강제 적용.

  3. 3

    편집 및 그룹핑

    화면에서 숨길 상품은 [빼기] (DB 삭제 아님). 알짜 상품만 [견적서 만들기]로 저장.

  4. 4

    마감

    F8 (마스터 확정) ➡️ F9 (통합 엑셀 다운) ➡️ F10 (이미지 추출)

[중요] 예외 상황 발생 시 대처법

🚨 1. 새로운 상품이 등장했을 때 (마스터 DB에 없음)

  • 하던 작업을 멈추고 ERP 메뉴로 이동해서 등록하고 온다.
  • 작업 화면 내에서 [ERP 코드로 직접 등록] 버튼 클릭. 즉석 코드가 부여되며 마스터 등록, 연산, 학습이 동시 처리됨.

🚨 2. 기존 취급 상품이 시세표에서 사라졌을 때 (단종 의심)

  • 절대 마스터 DB에서 '삭제(Delete)'하지 마십시오! (과거 정산 이력 훼손 우려)
  • [카탈로그 제외] 기능을 사용하세요. 원본은 보존되며 현재 화면과 해당 쇼핑몰에서만 안전하게 '숨김' 처리됩니다.

산출물 활용 가이드 (F9, F10)

[F9] 통합 엑셀 보고서

  • 1. 쇼핑몰 업로드 시트: B2B 쇼핑몰 표준 양식. 재고수량까지 맞춰져 있어 즉시 업로드 가능.
  • 2. ERP 업데이트 시트: 내부 재무/재고 자산 갱신용. 매입원가 및 소분 단가 정리.
  • 3. 화면 엑셀 변환 시트: 변경 전/후 단가 및 차액을 캡처하듯 저장한 대조/백업용.

[F10] 모바일 단가표 이미지

모바일 화면 비율에 최적화된 깔끔한 이미지로 자동 출력됩니다.

엑셀을 열어보기 힘든 영업 현장이나 식당 사장님들의 카카오톡/문자로 즉시 공유하여 영업력을 극대화하세요.
