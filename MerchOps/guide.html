MerchOps Merchandising Operations

초보자를 위한 친절한 사용 가이드

MerchOps는 엑셀 견적서와 카톡 시세표를 분석하여,
단가 변동을 추적하고 시장 트렌드 리포트를 발행하는 MD 전용 자동화 솔루션입니다.

Step 01 데이터 준비 Step 02 시세 파싱 Step 03 마진 설정 Step 04 리포트 발행

"MD님, 아직도 하나하나 가격을 고치시나요?"

수천 개의 상품 단가를 엑셀로 비교하고 카톡 내용을 옮겨 적는 시간은 이제 끝났습니다.
MerchOps는 복잡한 계산을 대신하고, MD님은 오직 의사결정에만 집중하도록 돕습니다.

01

마스터 데이터 세팅하기

우리의 기준이 되는 ERP 상품 목록을 먼저 등록해야 합니다.

엑셀 파일 직접 업로드

우측 상단의 원통 아이콘(🗄️)을 눌러 ERP에서 내려받은 전체 상품 목록을 업로드하세요.
자동으로 품목코드와 단가 정보가 메모리에 로드됩니다.

클라우드 동기화 (GAS)

환경설정에서 클라우드 URL을 등록했다면 구름 모양 아이콘을 통해 데이터를 실시간으로 백업하거나 동료가 작업한 최신 DB를 가져올 수 있습니다.

02

지능형 텍스트 파서 사용하기

카톡이나 웹페이지의 지저분한 시세표를 분석합니다.

복사 & 붙여넣기만 하세요.

[파서 뷰] 모드에서 [+] 버튼을 누르고 시세표 전문을 붙여넣으세요.
AI 엔진이 품명, 규격, 입고가를 자동으로 찾아냅니다.

매핑 팁 (Mapping)
  • 🟢 일치: 시스템이 확신하는 항목
  • 🟡 유사: MD의 확인이 필요한 항목
  • 🔴 불일치: 직접 연결이 필요한 항목
// AI 분석 예시
"국산 냉장 삼겹살 1kg / 18,500원"
-> 분석결과:
{
  name: "삼겹살",
  spec: "1kg",
  price: 18500
}
03

스마트 마진 자동 계산

입고가가 바뀌면 판매가는 시스템이 정한 규칙대로 바뀝니다.

하이브리드 마진 트리(Tree) 구조

Priority 1
창고 + 단위 일치

특정 창고의 특정 규격(Box 등)에만 개별 마진 적용

Priority 2
전체 창고(*) + 단위

창고 상관없이 모든 '낱개(ea)' 상품에 공통 마진 적용

Priority 3
기본값 (Default)

어떠한 조건에도 해당하지 않는 경우의 기본 마진

04

리포트 발행 및 이어달리기

최종 결과물을 엑셀로 뽑거나 웹 리포트로 전송합니다.

엑셀 보고서 출력 F9

ERP 업데이트용과 쇼핑몰 업로드용 양식이 하나의 파일에 담겨 출력됩니다.
단가가 0원인 상품의 품절 처리 여부도 선택 가능합니다.

웹 리포트 이어달리기

브리핑(F8) 창에서 [웹 리포트로 이어달리기]를 누르면 실제 판매가 변동 데이터가 Market Trend Report 페이지로 즉시 전송됩니다.

웹 리포트로 이어달리기
실시간 전송 시각화 분석

마스터 MD 필수 체크리스트

입고가 반영 전 확인

파싱된 입고가가 너무 낮거나 높지는 않은지 🟡 유사 탭에서 필히 검수하세요.

로컬 이력(History) 활용

작업 중 실수로 단가를 잘못 넣었다면 '이력' 버튼을 눌러 이전 가격을 확인하세요.

F3 검색 생활화

수천 개의 품목 중 원하는 상품을 찾을 땐 마우스를 쓰지 말고 F3을 누르세요.

클라우드 백업 주기

대량의 단가 반영 작업이 끝난 후에는 반드시 구름 아이콘(업로드)을 눌러 백업하세요.
